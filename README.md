# 📌 Project Overview

The Dubai residential real estate market is one of the world's fastest-growing property markets, generating thousands of property transactions across various communities each year.

This project transforms raw residential property transaction data into meaningful business insights using Microsoft Power BI.

The dashboard enables stakeholders to understand market performance, property pricing, buyer preferences, and sales trends through interactive visualizations.

---

# 🎯 Business Problem

Real estate stakeholders often face challenges in understanding:

- Which residential communities generate the highest sales?
- Which property types are most popular?
- How do property prices vary across locations?
- What market segments dominate the industry?
- How do monthly sales fluctuate?

This dashboard answers these business questions using data analytics.

---

# 🎯 Project Objectives

The primary objectives of this project were to:

✔ Analyze residential property sales.

# 📂 Dataset

The dataset includes residential property transactions containing:

| Column | Description |
|---------|-------------|
| Property Type | Apartment, Villa, Townhouse, Penthouse |
| Area Name | Residential Community |
| Property Value | Selling Price |
| Bedrooms | Number of Bedrooms |
| Area (Sq.ft) | Property Size |
| Price per Sq.ft | Price per Unit Area |
| Market Segment | Affordable, Mid-Market, Premium, Luxury |
| Project Name | Residential Development |
| Nearest Metro | Closest Metro Station |
| Nearest Mall | Nearby Shopping Mall |
| Transaction Date | Date of Sale |

# 📊 Key Business Insights

The analysis revealed that:

- Apartments accounted for the largest share of residential transactions.
- Premium residential communities generated the highest sales values.
- Luxury properties commanded significantly higher average prices than standard properties.
- Property prices increased with bedroom count and property size.
- Monthly transaction volumes fluctuated, indicating seasonal patterns in market activity.

---

# 💡 Recommendations

- Prioritize investment in consistently high-performing residential communities.
- Expand premium and luxury housing developments where demand is strongest.
- Use monthly sales trends to optimize marketing campaigns and launch periods.
- Monitor price-per-square-foot trends to improve pricing strategies.
